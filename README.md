# RBC News Parser

A simple Python parser that collects news from the main page of [RBC](https://rbc.ru) and saves them to a text file with links.

---

## Features

- Opens [RBC](https://rbc.ru) homepage.
- Searches for all news items displayed on the main page.
- Extracts news titles and their corresponding links.
- Saves the results to a formatted `news.txt` file.

---

## Requirements

- Python 3.x
- [Selenium](https://pypi.org/project/selenium/)
- [BeautifulSoup4](https://pypi.org/project/beautifulsoup4/)
- [time]
